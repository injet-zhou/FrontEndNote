#### 复合选择器

##### 1. 后代选择器

后代选择器可以选择**父元素**的**子元素**，这种父子关系可嵌套多层，使用时将最远的祖先元素放在最外面，父子元素之间用空格隔开，格式如下：`father-element child-element { style }`，示例：

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        ul li a {
            text-decoration: none;
            color: red;
        }
    </style>
</head>
<body>
    <ol>
        <li>我是ol的第1个孩子</li>
        <li>我是ol的第2个孩子</li>
        <li>我是ol的第3个孩子</li>
        <li>我是ol的第4个孩子</li>
        <li>我是ol的第5个孩子</li>
    </ol>
    <ul>
        <li>我是ul的第1个孩子</li>
        <li>我是ul的第2个孩子</li>
        <li>我是ul的第3个孩子</li>
        <li>我是ul的第4个孩子</li>
        <li>我是ul的第5个孩子</li>
        <li><a href="#">超链接</a></li>
    </ul>
</body>
</html>
```

##### 2. 子元素选择器

子元素选择器选择的是某元素**最近**的子元素，格式是`father-element > child-element { style }`，示例：

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Document</title>
    <style>
        div>a {
            color: red;
            text-decoration: none;
        }
    </style>
</head>
<body>
    <div>
        <a href="#">我是子元素</a>
        <p>
            <a href="#">我是孙元素</a>
        </p>
    </div>
</body>
</html>
```

##### 3. 并集选择器

并集选择器将一组选择器合并使用相同的属性，相当于把各个选择器的共同样式放在一起，格式: `selector_one,selector_two { style }`

##### 4. 伪类选择器

伪类选择器的作用是当某条件发生时修改样式，比如鼠标放在链接上时链接变颜色，就用到了`a:hover`，常见的伪类选择器有如下几种

| hover   | 鼠标放在某处时                     |
| ------- | ---------------------------------- |
| active  | 活动的链接（鼠标按下未弹起的链接） |
| visited | 访问过的链接                       |
| link    | 所有未访问的链接                   |

此外，链接的伪类选择器使用需要注意顺序，应为link、visited、hover、active，简记为LVHA，如果顺序不当可能无法生效。

##### focus伪类选择器
