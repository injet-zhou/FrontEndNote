#### 表单标签

表单由**表单域**、**表单元素**、**表单控件**组成，其中表单域是指用来收集信息用的区域，通常是`<form>`标签中的内容，`<form>`标签有三个属性，分别是action、method、name，action属性表示提交后服务器处理的程序，method表示提交的方法，如POST等。

##### `<input>`表单元素

input，输入的意思，可由type设置多种类型，有如下几种：

| button   | 可点击按钮     |
| -------- | -------------- |
| checkbox | 复选框         |
| file     | 浏览文件       |
| hidden   | 隐藏的输入字段 |
| image    | 图像形式的按钮 |
| password | 密码           |
| radio    | 单选按钮       |
| reset    | 重置按钮       |
| submit   | 提交按钮       |
| text     | 文本输入框     |

`<input>`标签有name、value、checked、maxlength等属性，name用来标识同一类表单元素，比如单选框和复选框要求有相同的name，value是用来设置输入框的值，checked表示选中。

##### `<label>`标签

`<label>`标签为`<input>`标签绑定标注，此时`<label>`的for属性值要与`<input>`标签的id或者class属性值相同。

##### `<select>`标签

`<select>`标签相当于一个单选按钮，但是可以容纳更多的选项，用法如下：

```html
<select>
    <option>选项1</option>
    <option>选项2</option>
    <option>选项3</option>
    <option>选项4</option>
</select>
```

需要注意的是`<select>`标签至少包含一个`<option>`标签，`<option>`标签的selected属性表示被选中。
