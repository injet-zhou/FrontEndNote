#### 溢出文字处理

##### 单行溢出省略号显示

```css
/* 强制一行内显示 不换行 */
white-space: nowrap;
/* 溢出的部分隐藏 */
overflow: hidden;
/* 文字用省略号待替超出部分 */
text-overflow: ellipsis;
```

##### 多行文本溢出省略号显示（有兼容性问题）

这种适用于`webkit`浏览器或移动端

```css
overflow: hidden;
text-overflow: ellipsis;
/* 弹性伸缩盒子模型显示 */
display: -webkit-box;
/* 限制在一个块元素显示的文本行数 */
-webkit-line-clamp: 2;
/* 设置或检索伸缩盒对象的子元素的排列方式 */
-webkit-box-orient: vertical;
```
