#### HTML5

##### H5新特性

HTML5是HTML目前最新的版本，针对以前的版本问题做了优化，新增了一写标签、新的表单和新的表单属性，虽然如此，新技术总有其缺点，比如新特性有**兼容性问题**，也就是说有一些IE版本（IE出来挨打）可能不支持这些新特性（IE9以上才支持，在IE9中需要把这些元素转换成**块级元素**）。

##### 语义化标签

以前网页设计都是用盒子（如div)来设计，对代码维护、SEO不是很友好，所以新增了一些语义化标签。

- `<header>`：头部标签
- `<nav>`：导航标签
- `<article>`：内容标签
- `<section>`：定义文档某个区域
- `<aside>`：侧边栏标签
- `<footer>`：尾部标签

```html
<body>
    <header>我是头部</header>
    <nav>导航栏</nav>
    <aside><section></section></aside>
    <aside></aside>
</body>
```

##### 多媒体标签

1. 视频：`<video>`
2. 音频：`<audio>`

对于视频，mp4格式的文件各浏览器的支持度更高，所以尽量使用mp4格式的。

兼容性写法:

```html
<video controls='controls'>
<source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
</video>
```

1. 自动播放`<video autoplay="autoplay">`，但是谷歌浏览器禁用了自动播放，此时需要用`muted="muted"`来搭配使用。
2. 循环播放：`loop="loop"`
3. 预加载：`preload="preload"`如果设置了自动播放，该属性自动禁用。
4. 当视频无法加载时，可以使用`poster`属性设置一张图片来待替。

##### 表单属性

| 属性           | 值          | 说明                                                         |
| -------------- | ----------- | ------------------------------------------------------------ |
| required       | required    | 表示拥有该属性的表单内容不能为空，必填                       |
| placeholder    | 提示文本    | 用来提示该表单的填写内容、格式等                             |
| `autofocus`    | `autofocus` | 自动聚焦，页面加载完成时自动聚焦到该表单                     |
| `autocomplete` | off/on      | 当用户在字段开始键入时，浏览器基于之前键入过的值，应该显示出在字段中填写的项，默认是打开的。 |
| multiple       | multiple    | 可以多选文件提交                                             |

可以使用伪元素对一些属性进行设置：`input::placeholder{color: red;}`

#### CSS3

##### 属性选择器

属性选择器可以根据元素特定的属性来进行选择，这样就可以不用借助类或者id选择器。使用方法：

| 选择符          | 简介                                        |
| --------------- | ------------------------------------------- |
| `E[att]`        | 选择具有`att`属性的E元素                    |
| `E[att="val"]`  | 选择具有`att`属性且属性值为'val'的E元素     |
| `E[att^="val"]` | 选择具有`att`属性且属性值以'val'开头的E元素 |
| `E[att$="val"]` | 选择具有`att`属性且属性值以'val'结尾的E元素 |
| `E[att*="val"]` | 选择具有`att`属性且属性值含有'val'的E元素   |

**注**：属性选择器的权重是10（类、伪类也是），比如`div[class^="att"]`的权重是11（div+[])

##### 结构伪类选择器

主要是根据文档结构来选择元素，常用于父级选择器里面的子元素。**权重为10**

| 选择符             | 简介                         |
| ------------------ | ---------------------------- |
| `E:first-child`    | 匹配父元素中的第一个子元素   |
| `E:last-child`     | 匹配父元素中的最后一个子元素 |
| `E:nth-child(n)`   | 匹配父元素中的第n个子元素    |
| `E:first-of-type`  | 指定类型E的第一个            |
| `E:last-of-type`   | 指定类型E的最后一个          |
| `E:nth-of-type(n)` | 指定类型E的第n个             |

上面的n可以是数字、关键字和公式（**even**偶数、**odd**奇数）。

案例

```html
<style>
        ul li:nth-child(2n) {
            color: red;  /* #1 */
        }
        
        ul li:nth-child(3n+1) {
            color: blue; /* #2 */
        }
        
        ul li:nth-child(3n+2) {
            color: pink; /* #3 */
        }
    </style>


<ul>
        <li>我是第1个孩子</li>
        <li>我是第2个孩子</li>
        <li>我是第3个孩子</li>
        <li>我是第4个孩子</li>
        <li>我是第5个孩子</li>
        <li>我是第6个孩子</li>
        <li>我是第7个孩子</li>
        <li>我是第8个孩子</li>
        <li>我是第9个孩子</li>
        <li>我是第10个孩子</li>
    </ul>
```

**注意**：n如果是公式，那么是从**0**开始的，如果是数字，那么是从1开始的。

###### 小技巧

n+5:选择从第五个开始的

-n+5:选择前五个

###### nth-of-type

作用跟nth-of-child类似，但是nth-of-child会把所有的盒子都排序号,执行的时候先看nth-of-child(1)，在下面的例子就是p标签，接着会看“：”前面的，下面的例子是div标签，这时候没有匹配上，就没有选出来。而nth-of-type先看**：前面的标签**，再看nth-of-type，再找第几个孩子。

```html
<style>
    section div:nth-of-child(1){
        color: red; /* not working */
    }
</style>
<section>
    <p>
        光头强
    </p>
	<div>
        熊大
    </div>
    <div>
        熊二
    </div>
</section>
```

##### 伪元素选择器

伪元素选择器可以帮助我们创建新的元素，而无需插入HTML标签，从而简化HTML结构。

1. ::before 在元素的内部的前面插入内容
2. ::after 在元素的内部的后面插入内容

伪元素选择器有以下特点：

1. 创建的元素属于行内元素
2. 创建的元素在文档树下是无法找到的。
3. 必须有`content`属性
4. 权重为1

伪元素一般搭配字体图标使用。
