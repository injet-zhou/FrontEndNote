##### 数组去重

暴力算法：

一个一个遍历，并把筛选出来的放到另外的素组

```js
function unique(arr){
    var newArr = [];
    for(var i=0;i<arr.length;i++){
        if(newArr.indexOf(arr[i]) === -1){ //元素不在新数组里
            newArr.push(arr[i]);
        }
    }
    return newArr;
}
```
