#### CSS三大特性

##### 👉层叠性

什么是层叠性先来看一段代码

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        div {
            color: red;
        }
        div {
            color: blue;
        }
    </style>
</head>
<body>
    <div>骑上我的小摩托</div>
</body>
</html>
```

我们给文字“骑上我的小摩托”设置了两个相同的属性，最后的效果会显示成什么样呢？最后的的效果就是文字的颜色变成了蓝色，所以层叠性就是当设置有冲突属性时实际所呈现的效果，即**样式覆盖**。它所遵循的原则是**就近原则**，就是样式越靠近结构，它的权重就越高，或者说越靠后的样式权重越高，后来者居上。

##### 👉继承性

继承就是子标签的样式继承父标签的样式，如text-，line-，font-等开头的属性一般会继承，以及color属性也会继承。

###### 行高的继承

行高可带单位，也可不带单位，不带单位则表示倍数，即字符大小的多少倍。如果子元素没有指定行高，则继承父元素的行高。

##### 👉优先级

优先级一般指的是选择器的优先级。优先级如下：

继承或者*<元素选择器<类选择器、伪类选择器<ID选择器<行内样式<important

总的原则就是选择的范围越小，权重就越大。

###### 权重叠加

复合选择器有权重叠加的问题，比如：

😻

```css
ul li {
    color: red;
}
```

和👻

```css
li{
    color: green;
}
```

实际效果会是😻而不是👻，因为0,0,0,1+0,0,0,1=0,0,0,2>0,0,0,1。

注：权重值如下

| 选择器     | 权重    |
| ---------- | ------- |
| 继承或者*  | 0,0,0,0 |
| 标签选择器 | 0,0,0,1 |
| 类或伪类   | 0,0,1,0 |
| ID选择器   | 0,1,0,0 |
| 行内样式   | 1,0,0,0 |
| important  | 无穷    |

==权重值没有进位==，`a:hover`的权重为0,0,1,1
